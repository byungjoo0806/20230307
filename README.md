# git 프로젝트

- git은 뭐냐?
1. 형상 관리 도구 중 하나
형산관리 도구 : 버전 관지 시스템
작업하면서 작업의 리스트를 관리 할수 있다.

- git의 장점
- 협업하는 단계에서 소스 코드를 파일로 주고 받을 필요가 없이
같은 파일을 팀원과 병렬로 작업이 가능하다.

- 눈으로 보고 찾는것보다 효율이 좋고 작업이 편하다.

- 코드의 다른 부분을 바로 찾을 수 있다.

- 경로 지정
- cd .. : 한 폴더 뒤로 이동
- ls -a : 경로의 파일을 전부 보고싶으면
- cd 폴더명 : 해당 폴더로 경로 이동
- cd 앞부분 폴더명 쓰고 기억이 안나면 tab 버트 눌러서 비슷한 폴더명 확인 가능
